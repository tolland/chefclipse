Chefclipse Cookbook Repository Discovery

Identifier:
org.limepepper.chefclipse.remotepicker.extpoint.org.limepepper.chefclipse.cookbook.repository

Since:
1.0.0

Description:
An extension point for contributing to the Chefclipse Cookbook Repositories.

Configuration Markup:

<!ELEMENT extension (repositoryDescriptor)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED

>


<!ELEMENT icon EMPTY>

<!ATTLIST icon

image32 CDATA #REQUIRED

>

A 32x32 image to show as the repository icon


  • image32 -

<!ELEMENT repositoryDescriptor (icon)>

<!ATTLIST repositoryDescriptor

id          CDATA #REQUIRED

name        CDATA #REQUIRED

description CDATA #REQUIRED

uri         CDATA #IMPLIED

class       CDATA #IMPLIED

builder     CDATA #IMPLIED

config      CDATA #IMPLIED

>

Description attributes for the repository of cookbooks. One or class or builder attributes must be specified.


  • id - Unique Id for each repository.
  • name - Repository Name
  • description - Repository Description to show in the remote picker wizard
  • uri -
  • class - A class implementing ICookbooksRepository for repositories that are static doesn't need parameters to be run.
  • builder - A class implementing ICookbooksRepository.Builder, responsable of configuring a repostiory with an UI and creating a ICookbooksRepository.
  • config - Optional paramenter for builder
